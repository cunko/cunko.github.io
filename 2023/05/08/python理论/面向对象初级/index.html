面向对象初级

发表于 2023-05-08 更新于 2023-07-03 分类于 python

python的第八节课笔记（基础高级）

借鉴函数的封装思维，再次对函数和变量进行封装，来提高代码的复用性，通过这样的封装，把和人相关的变量、函数等放在了一起，只要找到这个部分代码，就能知道关于人已经定义了哪些变量和函数，为了便于区分，把用关键字 class 定义的代码块称为 类

注意：

  • 在类里面的变量称为 类的属性
  • 在类里面的函数称为 类的方法
  • 本质就是使用面向对象进行封装—封装成一个类
  • 类定义的关键字： class
