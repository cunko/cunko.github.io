伪类选择器：

  • link 未访问时的状态
  • hover 鼠标悬浮时的状态
  • active 鼠标点击时的状态
  • visited 鼠标点击松开后的状态

优先级总结:

  • 复杂选择器

    id 个数不等，id越多，优先级越高
    id相同，class越多，优先级越高
    id，class都相同，则标签越多，优先级越高

  • 普通选择器 id > class > 标签

  • 都相同的情况下满足就近原则
