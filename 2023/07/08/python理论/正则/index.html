正则的概念

正则表达式：正则表达式由需要匹配的字符和一些特殊字符组成，可以在字符串当中匹配出需要查找的对象

匹配：通过正则表达式快速快捷的匹配出符合要求的特征字符串，比如利用病毒的特征值来查找文本中的病毒

元字符
