if条件语句和while循环

发表于 2022-09-24 更新于 2023-05-03 分类于 python

python的课程笔记

if 条件语句：

单项选择：

if 后面接的是判断条件:
满足条件的时候执行的代码
else:
不满足条件的时候执行的代码

缩进：

在Python中一般使用四个空格表示一个缩进 快捷键：tab
表示层级关系,有缩进的表示他是比他少一个缩进的一部分

多项选择：

以上代码的问题：黄灯和绿灯界限不够明显
原因：只有两种判断（红灯和其他）
解决：增加判断（红灯、绿、黄）

多中情况判断可以使用 if - elif - else
if 条件1：
满足条件1执行
elif 条件2：
满足条件2执行
………………
elif 条件n:
满足条件n执行
else:
以上条件全部不满足时执行

注意： if elif else 语句后面必须有冒号
里面的代码必须缩进
if 建议只有一个
elif 不限制个数
else只能有一个,并且不需要写判断条件

三目运算

条件成立执行 if 判断条件 else 条件不成立执行

if 嵌套

if 判断里面还有有一个if 判断

if 条件1：
满足条件1执行
if 条件：
条件满足执行
else:
条件不满足执行
else:
以上条件全部不满足时执行

input不管输入什么返回的都是字符串， 字符串和数值不能比较

逻辑运算符

and
or
not

优先级

not > and or
not False or False
先not 再其他（按先后顺序）

while循环

初始化条件值
while 条件：
满足条件要执行的代码
自增条件

不满足条件的时候，结束循环

跳出循环

手动提前跳出循环
break
跳过当前循环，其他循环按要求执行
continue

当循环正常结束的时候会执行else里面的代码
break是强制退出，不执行else的代码
continue是正常结束，执行else

只要是循环，
不管是while还是for
都可以和continue以及break

嵌套循环

1、先判断外层循环
2、外层条件满足，判断内层循环
3、内层循环结束，回到下一次外层循环
