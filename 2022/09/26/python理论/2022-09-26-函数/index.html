函数

发表于 2022-09-26 更新于 2023-05-03 分类于 python

python的课程笔记

def 用于定义函数的关键字
函数：一个可以重复使用的，用于实现单一或者相关功能的代码块

函数的定义语法

def 函数名():
函数体代码块(你想要完成的具体功能)

函数名() # 函数的调用
注意：函数只有在调用了之后才会执行

函数名和变量名一样都是标识符
命名规则一致

函数名和变量名一样，可以进行互相传递

需求： 完成两个数的相加，并且获取到结果

返回值：用于返回函数体里面的值
关键字：return
语法：
return 想要返回的值
返回的位置： 函数的调用处
特点：返回值可以使用变量接收
变量 = 函数名()

需求：求任意数的和

可变长参数

没有指定名称的参数
*args 以元组的形式进行接收
指定了名称的参数
**kwargs 以字典的形式接收多余的参数

指定名称：在函数的调用处以【变量名 = 值 】形式进行传参就叫做指定参数的名称

上面存在的BUG: 只能求固定值的和
目的：然函数变得更加灵活，可以求任意两个数的和
方法：给函数增加参数

必备参数： 一一对应原则
形参：变量的名字写在函数定义处
实参：变量的值写在函数的调用处
用实参给形参传递具体的值的过程称为–>传参

BUG: 在必备参数中，形参和实参个数不一致，会报错
需求：定义一个函数，将两数之和默认增加100（可以指定增加的值）之后返回

默认值参数：

在函数的定出使用【变量名=值】形式指定
不传值 不报错
传值 覆盖原来的
原理：变量名重复，覆盖原来的值
c = 100
print(c)
c = 30
print(c)

参数的顺序：

​ 必备参数、默认值参数、可变长参数
传参的顺序：
​ 指定名称的放在没有指定名称的后面
​ 不要和默认值参数名字一致

return 具有结束函数的功能，return后面的代码不再执行

一般情况下，看打到缩进就要想到作用域的概念
但是注意：if条件和循环没有作用域的概念

local：局部作用域，即函数中定义的变量
作用范围小 优先级越高
enclosing：嵌套的父级函数的局部作用域
相对外面来说是局部
相对嵌套的子函数来说，是一个全局
global：全局作用域 自己定义的，写在最外面的变量
作用范围大 优先级越小
build-in：内建作用域，不需要定义，可以直接拿来用的;

在函数里面更改外面的值：
声明全局变量： global
声明局部变量： nonlocal

闭包

1，存在于嵌套关系的函数中
2，嵌套的内部函数引用了外部函数的变量
3，嵌套的外部函数会将内部函数名作为返回值放回

作用：对于内部数据的保护

递归函数
