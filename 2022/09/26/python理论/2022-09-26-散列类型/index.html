散列类型

发表于 2022-09-26 更新于 2023-05-03 分类于 python

python的课程笔记

字典 dict

字典里面的元素，以键值对的形式存在

定义：

方式一：
{key:value, key2:value2}
key:value 键值对 （将键和值使用冒号链接就叫做键值对）
key 键，冒号前面的
value 值，冒号后面的

方式二： dict(变量名=值, 变量名1=值2)

要求：
键必须是不可变的数据类型(int、float、str、tuple)
值key是任意的数据类型

空字典:
{}
dict()

字典的查询

字典的键必须是唯一的
注意：如果健名重复，后面的值会覆盖掉原来的值

dict[key]
存在则返回键对应的值
不存在则报错

dict.get(key)
存在则返回键对应的值
不存在则返回None(空值)

字典的修改

dict[key]
key存在，新值覆盖旧值
key不存在，新建键值对

dict.setdefault(key, 值)
key 存在，返回所对应的value
key 不存在 新建键值对（键值对的值为指定在括号里面的第二个参数）

dict.update({k1:v1,k2:v2})
当健名重复的时候，可以进行一个批量替换

思路：
1、找到
2、赋予新的值覆盖掉旧值

字典的删除

dict.pop(key) 通过key删除键值对 key不存在将报错
dict.popitem() 随机删除,本质是随机删除，
但是python3.6版本之后，就自动做了一个排序，表现为删除最后一个
del dict[key]
del dict 删除整个字典

集合

集合的运算

交集 & 两个集合相同的元素
并集 | 两个集合全部加在一起
差集 - 我有你没有

集合的增删改查

添加单个 set.add(元素)
添加多个 set.update(可迭代对象)
随机删除 set.pop()
指定删除 set.remove()

阶段总结

判断的时候,判断的都是布尔值(True False)

如何判断布尔值—bool() 判断某一个数据的布尔值
数值:
0 为 False
其他为 True
其他数据类型:
空 False
但凡里面有值都是 True

六大基础数据类型总结
数值: int float bool
不可变的非序列
可以进行数学运算
字符串: str
不可变的序列
可以更具索引(下标)查找值
可以增删改查,但是要使用新的变量接受
元组: tuple
不可变的序列
没有增删改查
可以进行拆包
列表: list
可变序列
增删改查
字典: dict
可变的键值对映射结构—可以快速查找
增删改查
集合: set
可变的,无序求去重
增删改查
